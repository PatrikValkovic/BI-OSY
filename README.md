BI-OSY
======

Implementation of programs for BI-OSY (Operation systems) subject.

Author: Patrik Valkovic  
CTU (Czech Technical University in Prague)  
FIT (Faculty of Information Technology)  
Summer semester 2016/2017

The first task was implemention of Floyd-Warshall and Ford-Fulkerson algorithms in parallel. The approach of producer-consumer is used.

Second task is implementation of RAID5 storage.

Detailed informations are in task folders (czech language).
